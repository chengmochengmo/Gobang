## 多房间五子棋对战

#### 拉取项目

`git clone -b dev https://github.com/chengmochengmo/Gobang.git`

#### 安装模块

`npm install`

#### 本地运行

- 安装nodemon `npm install nodemon -g`
- 启动服务 `npm run serve`
- 启动项目 `npm run dev`

#### 部署

- 打包 `npm run build`
- 部署需要node环境 推荐pm2启动

#### 目录结构

- public 前端入口
- build 打包脚本
- serve 服务端
- web 客户端
- common 公共
